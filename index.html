PROFILE  //  001 CURRENTLY BUILDING OMELO

Amogh Tiwari

I build consumer software for a living, which means my job is to get people to open an app, come back, and eventually pay for it. That is also why I write about behavioral design: from inside the room where the decisions get made, not from above it.

Behavioral Debt is my attempt to map how software moves people, what it quietly costs them, and how to build the honest version. It is written in the open, and every claim in it is meant to be checked.

Currently building Omelo, precision pet healthcare and parenting
